---
name: Orval query hooks require queryKey
description: Every useQuery hook generated by Orval requires an explicit queryKey in the query options object, or TypeScript will error (TS2741).
---

Every `useXxx` query hook generated by Orval requires an explicit `queryKey` in the `query` options or TypeScript raises TS2741 ("Property 'queryKey' is missing").

**Why:** The generated `UseQueryOptions` type in this project makes `queryKey` required, not optional.

**How to apply:** Always pair the hook with its query key getter:
```ts
import { useGetProject, getGetProjectQueryKey } from "@workspace/api-client-react";

useGetProject(id, { query: { enabled: !!id, queryKey: getGetProjectQueryKey(id) } });
```
The getter name follows the pattern `getGet<Resource>QueryKey(args)`.
